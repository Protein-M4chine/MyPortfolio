BucBoard

Check out the code on GitHub!

The BucBoard is a centralized hub where professors can notify students of important announcements, and where students can schedule office meetings with those professors. This application features two main parts: (1) Touch screen functionality for professors to run this application on touch screens right outside there office for students to interact with and (2) Remote access for the professor which allows the professor to log into an online portal and update the announcements log.

This web application was developed in my Software Engineering 2 class by my team (Dreamteam 2.1 Beta) which consisted of myself, Austin Funderburke (Scrum Master), Charles Walker (Database Administrator/Back-end Developer), Chase Day (Project Owner/Front and Back-end Developer), Dan Slover (Metrics Analyst), and Tanner Smith (Front-end Developer/Presentation Organizer). I focused mainly on the back-end CRUD aspects of this application, particularly the announcement and schedule functions.

The framework utilized for this project was ASP.NET Core MVC

Online Supermarket

Check out the code on GitHub!

This is an academic project that I developed for my Advanced Web Development course my last semester at University. I created this web application with Microsoft's ASP.NET framework using MVC with Identity Account Authorization.

It features an online supermarket where shoppers can buy grocery items with full shopping cart functionality. There are two types of accounts: admin, or standard user. Admins can sells items (they have CRUD permissions for the items) while standard users can purchase items through the shopping cart checkout.
